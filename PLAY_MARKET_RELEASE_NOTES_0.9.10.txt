What's new in 0.9.10

- Added current app version display on the Settings screen.
- Restored Google Play update checking from Settings with clear success/failure messages.
- Added transfers between own cash accounts with a clear Payment / Transfer selector.
- Transfers are stored as linked entries without categories and are excluded from income/spending reports.
- Added database migration and integrity checks for the new payment type table.
- Improved add-category and add-subcategory dialogs: cleaner layout, better field sizing, and more consistent spacing.
- Fixed add actions on the category screen so the whole row can be tapped, not only the plus icon.
- Renamed category actions to make categories and subcategories clearer.
- Reworked first-launch currency selection: currencies are now selected with checkboxes and grouped into major currencies, other currencies, and cryptocurrencies.
- Updated first-launch currency dialogs and currency rows: symbol, code, and name are shown compactly in one line.
- Refined starter currency names to make the list easier to read for Russian, English, and European users.
- Replaced the broken borrowed calculator with a simple payment calculator and a two-entry calculation history.
- Added more fiat currencies and optional cryptocurrency choices.
- Added text symbols for major cryptocurrencies where they render reliably.
- Added confirmation when a cryptocurrency is selected as the default currency.
- Updated input fields and dialog buttons: cleaner outline styling, darker text on white buttons, and better aligned paired buttons in dialogs.

Что нового в 0.9.10

- Добавлены переводы между своими счетами с понятным переключателем Платёж / Перевод.
- Переводы сохраняются связанными записями без категории и не попадают в отчёты доходов/расходов.
- Добавлена миграция БД и проверки целостности для новой таблицы типов платежей.
- Улучшены диалоги добавления категории и подкатегории: аккуратнее расположены поля, размеры и отступы.
- Исправлены действия добавления на экране категорий: теперь нажимается вся строка, а не только значок плюса.
- Переименованы действия добавления, чтобы понятнее разделять категории и подкатегории.
- Переработан выбор валют при первом запуске: теперь валюты выбираются флажками и разделены на основные, остальные и криптовалюты.
- Обновлен дизайн диалогов первого выбора валют и строк валют: символ, код и название отображаются компактно в одну строку.
- Уточнены названия стартовых валют, чтобы список был понятнее для русско/англо/европейской аудитории.
- Сломанный старый калькулятор заменен простым калькулятором для платежей с историей на два вычисления.
- Добавлены дополнительные валюты и отдельный список популярных криптовалют.
- Добавлены текстовые символы для основных криптовалют, где они корректно отображаются.
- Добавлено подтверждение, если криптовалюта выбирается основной валютой.
- Обновлены поля ввода и кнопки диалогов: аккуратный outline-стиль, более темный текст на белых кнопках и стабильное выравнивание парных кнопок.
