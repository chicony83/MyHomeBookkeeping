What's new in 0.9.9

- Fixed category screen list glitches: items no longer overlap while scrolling.
- Added category search from the top app bar.
- Improved filtering for categories and parent categories.
- Improved parent category row layout so long names wrap more cleanly.
- Fixed creating and editing categories without a parent category.
- Improved first-launch currency selection handling.
- Updated the app to target Android SDK 35.
- Added regression tests for categories without a parent category.
